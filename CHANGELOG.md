# Changelog

## 10.0.0 — Workflow-centred UI

### Added
- Role-aware My Worklist
- Global patient/operation/appointment/result search
- Quick-add modal
- Longitudinal Patient Chart
- Persistent selected-patient and selected-operation context
- Surgical workflow stepper
- Theatre list/calendar toggle
- Action-oriented Results Inbox
- Ward Board filters and quick observation workspace
- Mobile/iPad quick navigation
- Database helpers for search, patient snapshot and patient-specific operations

### Changed
- Replaced the large hero with a compact hospital-style application shell
- Grouped navigation by Workspace, Clinic, Surgery and Clinical tools
- Reduced card sizes and improved information density
- Improved responsive behaviour and touch targets
- Updated Streamlit components to the current `width` API

### Compatibility
- No new database tables
- Compatible with the v9 SQLAlchemy schema
